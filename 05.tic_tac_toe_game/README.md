# 🎮 React Tic Tac Toe Game

A classic Tic Tac Toe game built with React. The application allows two players to compete with automatic win detection, draw handling, and game reset functionality.

## Features

- Two-Player Gameplay
- Automatic Win Detection
- Draw Detection
- Reset Game Functionality
- Interactive UI

## Tech Stack

- React
- JavaScript
- CSS

## Purpose

This project was built as a learning exercise to strengthen my understanding of React state management, refs, event handling, conditional logic, and building interactive applications.
