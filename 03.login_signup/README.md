# 🔐 React Login & Signup Page

A responsive Login & Signup interface built with React to practice core React concepts. The application allows users to switch seamlessly between Login and Signup forms using React state and conditional rendering.

## Features

- Login & Signup Toggle
- Conditional Rendering
- State Management with `useState`
- Responsive UI

## Tech Stack

- React
- JavaScript
- CSS

## Purpose

This project was built as a learning exercise to strengthen my understanding of React state management, conditional rendering, reusable components, and interactive UI development.
