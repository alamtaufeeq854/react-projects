# 🧠 React Quiz App

An interactive Quiz application built with React to practice core React concepts. Users can answer multiple-choice questions, receive instant feedback, track their score, and restart the quiz.

## Features

- Multiple Choice Questions
- Instant Answer Validation
- Score Tracking
- Question Navigation
- Restart Quiz Functionality

## Tech Stack

- React
- JavaScript
- CSS

## Purpose

This project was built as a learning exercise to strengthen my understanding of React state management, refs, conditional rendering, event handling, and building interactive user interfaces.
