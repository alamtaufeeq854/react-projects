# ChatGPT Clone

A ChatGPT-inspired AI chatbot built with React and the Google Gemini API.

## 🚀 Features

- AI-powered conversations
- Real-time responses
- Chat history during the session
- ChatGPT-inspired responsive UI
- Suggested prompts for quick interactions

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- Google Gemini API
- React Markdown

## ▶️ Run Locally

```bash
npm install
npm start
```
