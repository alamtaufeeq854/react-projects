# 📝 React Todo App

A simple Todo application built with React to practice core React concepts. Users can add, complete, and delete tasks, with data stored in the browser using Local Storage for persistence.

## Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage with Local Storage

## Tech Stack

- React
- JavaScript
- CSS
- Local Storage API

## Purpose

This project was built as a learning exercise to strengthen my understanding of React state management, component-based development, and browser storage.
