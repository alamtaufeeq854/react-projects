# AI Image Generator

An AI-powered image generator built with React and the Pollinations API.

## 🚀 Features

- Generate images from text prompts
- Fast AI-powered image generation
- Responsive and clean UI
- Default placeholder image
- Error handling for failed requests

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- Pollinations AI API
- Vite

## ▶️ Run Locally

```bash
npm install
npm run dev
```
