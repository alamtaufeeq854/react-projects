# 🌐 React Landing Page

A modern and responsive landing page built with React. The project features a dynamic hero section with image and video backgrounds, interactive navigation, and reusable components.

## Features

* Responsive Landing Page
* Dynamic Hero Section
* Image & Video Background Toggle
* Reusable Components
* State Management with `useState`

## Tech Stack

* React
* JavaScript
* CSS

## Purpose

This project was built as a learning exercise to strengthen my understanding of React component composition, state management, conditional rendering, and creating interactive user interfaces.
