# ✨ React Gemini Clone

A Gemini-inspired AI assistant built with React that generates real-time responses using the Google Gemini API.

## Features

- Google Gemini API Integration
- AI-powered Conversations
- Real-time Response Generation
- Suggested Prompt Cards
- Recent Chat History
- New Chat Functionality
- Responsive UI
- Context API for State Management

## Tech Stack

- React
- JavaScript
- CSS
- Google Gemini API

## Purpose

This project was built as a learning exercise to strengthen my understanding of Context API, global state management, AI API integration, asynchronous programming, and building interactive AI-powered applications.
